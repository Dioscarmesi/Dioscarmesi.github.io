About the Project

Furry Spanish es un mundo social multi‑zona dentro de VRChat enfocado a la comunidad hispanohablante. El proyecto integra environment art con dirección técnica y sistemas modulares en UdonSharp. Incluye una capa de roles y permisos impulsada por JSON externo (GitHub) para administrar accesos sin reconstruir el mundo, espacios listos para eventos (karaoke y zonas sociales), y controles orientados a estabilidad para mantener una experiencia consistente en PC, Quest e iOS.

Core Systems

  • Roles & Permissions (JSON via GitHub): feature gates para staff/VIP y herramientas de comunidad sin rebuild.
  • Cross‑Platform Performance Pipeline: estrategia PC/Quest/iOS (texturas, lightmaps, batching/instancing, LOD/occlusion, materiales).
  • Safety & Anti‑Abuse Tooling (alto nivel): sistemas protegidos, restricciones de interacción y fallbacks de estabilidad.
  • Voice Zones + Directional Audio & Occlusion: control acústico para convivencia y eventos.
  • Media Hangout Space (ProTV): sala pública de reproducción + convivencia.
  • Karaoke Room & System: sala dedicada con UX de cola/búsqueda y operación para eventos.
  • Patreon Personalization (Moderated): posters personalizables para supporters mediante un flujo curado/moderado.

Responsibilities

Environment / Technical Art

  • Diseño de entornos multi‑zona para sesiones largas y circulación durante eventos.
  • Estrategia de rendimiento cross‑platform (PC/Quest/iOS): budgets de texturas, lightmaps, instancing/batching, LOD/occlusion y materiales por plataforma.
  • Mantenimiento de identidad visual premium respetando límites de runtime para standalone.

Systems / UdonSharp

  • Sistema de roles/permisos por JSON (GitHub) para actualizar reglas de acceso sin rebuilds.
  • Feature gating y herramientas de operación de eventos con restricciones orientadas a estabilidad.
  • Implementación de voice zones con audio direccional y oclusión.

Community / Operations

  • Soporte a convivencia diaria y eventos especiales mediante flujos operativos para staff.
